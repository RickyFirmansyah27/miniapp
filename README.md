# OJT DAnS Multi Pro - React Redux MiniApp

## Install Package
### Instalasi semua package yang diperlukan dalam menjalankan miniapp
```bash
npm install
```
---------

## Run
### Ada dua command yang harus di run  yaitu:
```bash
npm start
```
```bash
json-server -w db.json -p 5000
```
